CliverRoutines

CliverRoutines is a cross-platform C# lib providing what is needed in most desktop applications and not only them:
- settings manager that supersedes .NET ConfigurationManager;
- logger with multi-threading and session support;
- auxiliary routines;

Documentation | Source code

CliverExcel

CliverExcel is an .NET API for creating and editing excel files.
It is based on the NPOI lib and, besides its own API, allows accessing the NPOI API, too. Its goal, however, is to provide an API more advanced than NPOI's.
